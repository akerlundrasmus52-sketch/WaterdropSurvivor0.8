# 🎮 Water Drop Survivor - Quick Start Guide

## 🚀 Play Immediately

### Desktop
```bash
# Just open the file!
open index.html
```
**Controls:**
- **Move:** WASD or Arrow keys
- **Aim:** Mouse
- **Dash:** Spacebar
- **Pause:** ESC or click ⚙ Menu

### Mobile
1. Open `index.html` in your mobile browser
2. Use the on-screen joystick to move
3. Tap right side to aim (or use auto-aim)
4. Swipe to dash

### Web Server (Optional)
```bash
python3 -m http.server 8000
# Then visit: http://localhost:8000
```

## 💡 Tips

1. **Collect EXP gems** to level up (they auto-collect when close)
2. **Choose upgrades wisely** - some are better than others
3. **Unlock weapons** at levels 5 and 10
4. **Watch out for mini-bosses** at levels 10, 25, and 50
5. **Use dash** to escape dangerous situations
6. **Vampire perk** at level 12/18/25 heals you with each hit
